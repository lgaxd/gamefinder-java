# GameFinder

Projeto desenvolvido como entrega do Projeto Rubi da turma 2TDSPX.

## Descrição

O GameFinder é uma Web API para busca de jogos, desenvolvida com:

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Spring HATEOAS

A aplicação disponibiliza endpoints para consulta de jogos, gêneros e plataformas.

## Endpoints
- GET /games
- GET /games/{id}
- GET /games/genres/{genreId}
- GET /games/platforms/{platformId}
